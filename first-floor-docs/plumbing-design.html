8. Contractor Notes / ठेकेदार के लिए निर्देश

Phase 1: Before Wall Construction / दीवार बनाने से पहले

  • Slab Sleeve Check (स्लैब स्लीव जांच): Verify if 4" pipe sleeve exists on RIGHT side of slab. If not, mark and core-cut a 4" (110mm) hole BEFORE starting walls. Also need 2" sleeve for kitchen waste if separate.
  • Soil Stack Position (सॉइल पाइप स्थान): Mark exact position of 4" vertical soil pipe on the slab — should be against right wall, close to where WC will be.
  • Floor Trap Positions (फ्लोर ट्रैप): Set nahani traps in concrete/screed BEFORE floor tiling. Bathroom floor must slope toward drain (1:100 minimum).
  • Drainage Pipe Below Slab: Run horizontal waste pipes (2") from kitchen and bathroom to the soil stack position BEFORE screeding the floor (if pipes run in floor). Otherwise, run them below the slab soffit (exposed/in false ceiling at GF).

Phase 2: During Wall Construction / दीवार बनाते समय

  • Pipe Chases (पाइप की नाली): Leave vertical chases in walls for concealed supply pipes — 50mm wide × 40mm deep for 1/2" pipe; 60mm wide × 50mm deep for 3/4" pipe. Do NOT chase after building — weakens wall.
  • Concealed Cistern Space: Build partition wall behind WC at minimum 120mm thickness (or use double-wall technique) to house concealed flush tank. Frame the mounting plate at correct height (800-1000mm from FFL).
  • Geyser Wall: Use solid block/brick behind geyser location. Need 4 nos heavy-duty wall plugs at 1800mm height. Also run concealed 6mm² power cable to a 15A switch at 2000mm.
  • Kitchen Wall: Leave chase for hot + cold supply pipes behind sink location.

Phase 3: After Walls, Before Plastering / प्लास्टर से पहले

  • Pipe Installation (पाइप डालना): Fix all CPVC supply pipes in wall chases. Test entire system at 2× working pressure (6 kg/cm² for min 2 hours) before covering.
  • Pressure Test (प्रेशर टेस्ट): MANDATORY — fill system with water, pressurize to 6 bar, hold for 2 hours. No drop = pass. Mark any leaks, fix, and re-test.
  • Photo Documentation: Take photos of ALL concealed pipe routes before plastering — will be needed later for any maintenance.
  • Clip Pipes: Secure all pipes with wall clamps every 600mm (horizontal) and 1000mm (vertical) before plastering over.
  • Cover Chases: Fill chases with chicken mesh + cement mortar after pipe testing passes.

Phase 4: After Plastering, Before Tiling / टाइल्स से पहले

  • Floor Slopes: Bathroom floor must slope toward nahani trap — minimum 12mm fall per meter (1:80). Verify with water test before tiling.
  • Waterproofing: Complete bathroom waterproofing (floor + walls 1ft above floor) AFTER drainage pipe installation and BEFORE tiling.
  • Fixture Points: Mark and verify all fixture outlet points match the chosen fixtures (height, spacing between H+C points = 150mm standard for mixers).

Phase 5: After Tiling / टाइल्स के बाद

  • Fixture Installation: Install WC, basin, shower fittings, kitchen sink, balcony tap.
  • Seal All Joints: Silicone sealant around WC base, basin wall junction, and kitchen sink edges.
  • Final Testing: Run all fixtures simultaneously — check flow, drainage speed, any leaks. Run for 30 minutes minimum.
  • Grease Trap: Install under kitchen counter after sink is fixed. Ensure easy access for cleaning.
IMPORTANT CHECKS / महत्वपूर्ण जांच:
  • All CPVC joints must use ONE-STEP solvent cement (not two-step primer). Allow 24hr cure before pressure test.
  • PVC SWR joints: use SWR solvent cement (different from CPVC). Check ring-seal rubber is in place for push-fit joints.
  • Drainage slope: soil pipe 1:40 minimum, waste pipe 1:60 minimum. USE SPIRIT LEVEL.
  • Vent pipe MUST extend minimum 150mm above roof level with cowl. Do not terminate inside building.
  • Keep cleaning eyes (access plugs) accessible — do not plaster/tile over them.
Coordination / तालमेल:
  • Plumber + Mason: Chase cutting must be done during wall construction, NOT after.
  • Plumber + Electrician: Geyser power point (15A) and instant heater point (20A) at correct locations BEFORE plastering.
  • Plumber + Waterproofing: Complete drain installations BEFORE waterproofing. Do NOT puncture waterproofing membrane after application.
  • Plumber + Tiler: Coordinate fixture outlet heights with tile layout (cuts, patterns).
